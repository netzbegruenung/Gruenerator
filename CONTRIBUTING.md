# Beitragende Richtlinien

Vielen Dank, dass du dich für eine Mitwirkung an Grünerator interessierst! Wir schätzen deine Hilfe und möchten sicherstellen, dass der Beitrag reibungslos verläuft.

## Wie man beiträgt

1. **Fork das Repository**: Erstelle eine Kopie dieses Repositories auf deinem GitHub-Konto.
2. **Erstelle einen Branch**: Erstelle einen neuen Branch für deine Änderungen.
   ```sh
   git checkout -b feature/Dein-Feature-Name
   ```
3. **Nimm Änderungen vor**: Führe deine Änderungen im neuen Branch durch.
4. **Teste deine Änderungen**: Stelle sicher, dass deine Änderungen ordnungsgemäß funktionieren.
5. **Commit deine Änderungen**: Schreibe sinnvolle Commit-Nachrichten.
   ```sh
   git commit -m "Beschreibung der Änderungen"
   ```
6. **Push den Branch**: Schicke deine Änderungen an dein GitHub-Repository.
   ```sh
   git push origin feature/Dein-Feature-Name
   ```
7. **Erstelle einen Pull Request**: Stelle einen Pull Request zu unserem Repository, beschreibe deine Änderungen im Detail und warum sie nützlich sind.

## Lizenz

Durch das Einreichen eines Beitrags erklärst du dich damit einverstanden, dass dein Beitrag dem Autor gehört und alle Rechte vorbehalten sind.
